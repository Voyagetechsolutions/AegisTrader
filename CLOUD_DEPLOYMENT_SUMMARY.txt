QUICK START (5 MINUTES)
================================================================================

OPTION 1: Render.com (Easiest)
1. Go to https://render.com
2. Sign up with GitHub
3. Click "New Web Service"
4. Select AegisTrader repo
5. Use build: pip install -r backend/requirements.txt
6. Use start: python -m uvicorn backend.main:app --host 0.0.0.0 --port 8000
7. Add environment variables (see DEPLOY_NOW.md)
8. Click "Create Web Service"
9. Wait 2-3 minutes
10. Get your URL: https://aegis-trader-backend.onrender.com

OPTION 2: Railway.app (Better)
1. Go to https://railway.app
2. Sign up with GitHub
3. Click "New Project"
4. Select "Deploy from GitHub repo"
5. Choose AegisTrader
6. Railway auto-configures
7. Add environment variables
8. Wait 2-3 minutes
9. Get your URL from dashboard

================================================================================
AFTER DEPLOYMENT
================================================================================

1. Test Backend
   curl https://your-backend-url/health

2. Update Mobile App
   Edit mobile/services/api.ts
   Change: const API_BASE_URL = 'http://192.168.8.152:8000';
   To: const API_BASE_URL = 'https://your-backend-url';

3. Rebuild Mobile App
   cd mobile
   npm run build

4. Keep Backend Awake (Render Only)
   Run: python keep_alive.py
   Or set up cron job at https://cron-job.org

IMPORTANT NOTES
================================================================================

✓ MT5 Connection
  - MT5 terminal must run on your local machine
  - Cloud backend connects to local MT5 via heartbeat
  - Your local machine must stay online
  - Network connectivity must be maintained

✓ Database
  - SQLite database stored on cloud
  - Data persists between deployments
  - Automatic backups recommended

✓ Costs

✓ Performance

================================================================================
TROUBLESHOOTING
================================================================================

Backend won't deploy:
- Check logs in platform dashboard
- Verify requirements.txt exists
- Check Python version (3.11+)

Mobile app can't connect:
- Verify cloud URL is correct
- Test URL in browser first
- Check CORS settings

Backend keeps spinning down (Render):
- Set up keep-alive cron job
- Or run keep_alive.py locally

MT5 connection fails:
- Ensure local MT5 is running
- Check network connectivity
- Verify heartbeat endpoint works
